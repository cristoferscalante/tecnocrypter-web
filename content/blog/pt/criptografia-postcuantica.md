# Criptografia pós-quântica: é assim que preparam algoritmos para o futuro

Os computadores quânticos representam uma ameaça real aos sistemas de criptografia atuais. Algoritmos como RSA e ECC podem ser quebrados em segundos com potência quântica suficiente.

## O que é criptografia pós-quântica?

São algoritmos projetados para resistir a ataques de computadores quânticos, usando matemática não vulnerável a Shor ou Grover.

## Projetos e padrões

- **NIST** lidera a padronização de novos algoritmos.
- **Kyber**, **Dilithium**, **SPHINCS+** são alguns dos selecionados.

## Aplicações futuras

- Comunicações governamentais.
- Bancos e sistemas críticos.

## Conclusão

Preparar-se hoje com algoritmos pós-quânticos é fundamental para um futuro digital seguro. A transição começou.
